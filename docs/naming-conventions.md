# Naming Conventions

All file names in this repository must use **kebab-case**: all lowercase letters with words separated by hyphens (e.g., `my-file.js`).

Exceptions:

- Standard documentation files like `README.md` or `CHANGELOG.md`.
- Auto-generated or third-party files that require a different naming scheme (e.g., Supabase migrations, Husky hooks).

When adding or renaming files, update any imports, references, and links to match the new file name.
